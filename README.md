# About this app
It's a simple story dice (icons) roller.
Features:
- tell a story
- change lang (English or Polish)
- change theme (dark and light)

# Run locally
## Requirements
- Node.js > 16
- Yarn
## How to run?
1. Download repo.
2. Go to the main folder.
3. Run `yarn install`.
4. Run `yarn start` to run app in the dev mode.
5. App is available under `localhost:3000`.
